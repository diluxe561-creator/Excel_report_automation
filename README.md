# Excel_report_automation

This Python script reads an Excel file with sales data,
calculates the total sales, and writes the summary to a new Excel file.

## Skills Used
- Python
- Excel Automation (openpyxl)
- File Handling

## How to Run
1. Place `sales_data.xlsx` in the project folder
2. Run:
   python excel_report.py
3. Check `sales_summary.xlsx` for results
